# Burlington Odd Fellows — 2026 Dividend ETF Screen

**Burlington Odd Fellows Assoc. Inc. — Board of Finance**

Internal review site for the 2026 dividend ETF screening process. Evaluates low-volatility, values-screened equity candidates for a $35,000 allocation.

## Purpose

This site presents the Board of Finance with a side-by-side comparison of seven ETF candidates screened against tobacco, alcohol, and petroleum exclusion criteria, alongside financial metrics (yield, beta, expense ratio, 2022 drawdown performance).

**Staff recommendation:** NULV (Nuveen ESG Large-Cap Value ETF) — the only candidate satisfying all hard constraints (tobacco + alcohol exclusions) while maintaining a genuine value/dividend tilt and adequate liquidity.
